# Control My Window

A GNOME Shell extension to help you manage your windows effectively.

## Features
- **Auto Hide**: Automatically minimize tracked applications when they lose focus.
- **Auto Focus**: Focus selected applications when they demand attention instead of showing a "window is ready" notification.
- **Remember**: Remembers the X/Y coordinates and workspace of your windows across sessions.
- **All Work**: Keep specific applications on all workspaces (Pinning).

## Requirements
- GNOME Shell 50 or later.

## install extension
```
bash install.sh
```
